pyziabm

A Zero-Intelligence Agent-Based Model of a Modern Limit Order Market

Overview

The code in this GitHub repository was used to generate the results in "Increasing the minimum price increment boosts market-maker profitability" by Charles Collver.

You can download or fork the code and run it yourself

Or, you can use one of the conda packages by downloading and installing or by: conda create -c dennycrane pyziabm. For now, there are Python3.6 versions in Windows, Unix and Mac.
